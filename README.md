# 🎓 Student Result Dashboard

A Student Result Analysis Dashboard built using Python, Pandas, Matplotlib, and Streamlit.

## 🚀 Features

- CSV dataset handling
- Dataset merging
- Student average calculation
- Grade classification system
- Top 3 student leaderboard
- Pass/Fail analysis
- Bar chart visualization
- Pie chart visualization
- Histogram analysis
- Interactive Streamlit dashboard

## 🛠 Technologies Used

- Python
- Pandas
- Matplotlib
- Streamlit

## ▶️ Run Project

```bash
streamlit run app.py
